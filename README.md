# Nones-API

Planetary API Observatory - An interactive website for exploring celestial data.

## Features

- Search and explore planetary information
- Interactive visualization with responsive design
- Fetch live data from planetary APIs
- Responsive card-based layout using Bootstrap
- Beautiful space-themed UI with glassmorphism effects

## How to Run

1. Open `index.html` in your browser.
2. Explore the planetary observatory interface.
3. View detailed information about celestial bodies.

## Technologies Used

- HTML5 & CSS3 (with modern animations and gradients)
- JavaScript for interactive features
- Bootstrap 5.3 for responsive design
- Public Planetary APIs

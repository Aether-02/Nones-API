# Nones-API

Avatar-inspired API-powered website with search functionality.

## Features

- Search Avatar characters by name
- Filter by nation affiliation
- Fetch live data from `last-airbender-api`
- Responsive card-based layout using Bootstrap

## How to Run

1. Open `index.html` in your browser.
2. Enter a character name and/or choose a nation.
3. Click **Search** to view matching characters.

## API Used

- [Last Airbender API](https://last-airbender-api.fly.dev/)
